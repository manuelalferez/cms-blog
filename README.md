## Installation

1. **Clone** the project.

   ```bash
   git clone https://github.com/manuelalferez/blog.git
   ```

2. **Change directory**.

   ```bash
   cd blog
   ```

3. **Install packages**.

   ```bash
   npm i
   ```

4. **Run** the project.

   ```bash
   npm run dev
   ```
